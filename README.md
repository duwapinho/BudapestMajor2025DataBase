# BudapestMajor2025DataBase
Data about Counter Strike 2 Budapest Major 2025.  
Sources: https://liquipedia.net/, https://www.hltv.org/.

PowerBi Dashboard.:

Interactive Power BI dashboard analyzing CS2 Major 2025, including.: 

PlayersPage: This page focuses on individual player performance during the tournament.
Featuring: average rating, K/D ratio, average damage per round (ADR), recent match performance (last 5 maps) and basic information such as Birthdate name nationality etc.

TeamsPage: This page show complete overview of a selected team's performance.
Featuring:match results against opponents, stage progression and final placement, prize money earned

Players Stats: This page highlights the best and worst performing players based on selected metrics. You can filter by stage, and players position.

Maps statistics This page provides an overview of team performance across different maps played during the tournament.
Featuring: number of maps played, number of wins, winrate per map, best performing map, worst performing map, most frequently played map
